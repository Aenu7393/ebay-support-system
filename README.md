
1. タイトル
EC在庫管理支援Webアプリ


2. 開発背景
複数のECサイトに出品している商品の在庫管理を効率化するため、個人利用を目的として開発した。  
同時に、Djangoを用いたWebアプリケーション開発や外部サービス連携、非同期処理設計などの学習を目的として実装を行った。


3. 概要
複数のECサイトに出品している商品の在庫状態を一元管理するWebアプリ。
複数の出品情報をGoogleスプレッドシートに集約し、
eBay出品との紐づけ管理を行うことで在庫切れ時の出品停止などを行う。


4. 使用技術
- Python
- Django
- PostgreSQL（本番環境）
- SQLite（開発環境）
- Google Sheets API
- Celery
- Redis
- Heroku


5. 主な機能
- ユーザー登録 / ログイン機能
- Googleスプレッドシートの自動生成
- ECサイト商品情報の管理
- eBay商品との紐づけ
- 在庫状況の同期管理
- バッチ処理による定期更新


6. 実行方法
本アプリは Heroku 上での本番運用を想定して実装しており、一部機能は本番環境のデータベースや外部サービス連携に依存している。  
そのため、ローカル環境では一部機能が正常に動作しない。
